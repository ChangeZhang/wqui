教程内容

本教程是创建一个非常简单的”新建待办事项”页面，可以填写待办事项的名称、时间和补充说明，可以显示创建结果。
此页面共涉及到了三个组件：面包屑、提示消息、表单。
最终页面效果见 新建待办事项。

第一步，下载文件

SUI 文件主要有三个，分别是：

  • sui.css核心组件的样式文件，必须。
  • sui-responsive.css响应式布局，可选。
  • sui-extends.css拓展组件样式，可选。
  • sui-all.css全部样式，包括三部分：sui.less, sui-responsive.less, sui-extends.less。
  • sui.js核心组件。因为sui的js组件依赖jquery，所以必须在sui.js之前引入jquery。
  • sui-extends.js拓展组件，可选。
  • sui-all.js包含了全部的核心组件和拓展组件。
其中除了sui.css是必须的，其他两个文件可以根据自己的需求选择。
上述的所有文件都有对应的压缩版，分别是 sui.min.css,sui-responsive.min.css,sui.min.js,sui-extends.min.js,sui-all.min.js。

引用CDN文件

最简单使用方式是直接引用我们的CDN文件:

  • g.tbcdn.cn/sj/dpl/1.0.0/css/sui.min.css
  • g.tbcdn.cn/sj/dpl/1.0.0/css/sui-responsive.min.css
  • g.tbcdn.cn/sj/dpl/1.0.0/js/sui.min.js

下载文件

如果不想引用CDN文件，可以直接去gitlab上下载源代码，也可以下载打包后的ZIP文件。

  • sui-source.zip
  • sui-min.zip

第二部，引用文件

在head中依次引入sui.css, sui-responsive.css和sui.js文件，代码如下：

<!DOCTYPE html>
<html>
<head>
  <title>新建待办事项</title>
  <meta charset="utf-8">
  <meta name="viewport" content="width=device-width, initial-scale=1.0">
  <meta name="description" content="">
  <link href="sui/css/sui.css" rel="stylesheet">
  <link href="sui/css/sui-responsive.css" rel="stylesheet">
  <script type="text/javascript" src="http://g.tbcdn.cn/sj/lib/jquery/dist/jquery.min.js"></script>
  <script type="text/javascript" src="sui/js/sui.js"></script>
</head>
<body>
</body>
</html>
